MameBlock.js

シンプルなビジュアルコード編集ライブラリ

  • シンプル: ほんの100行のJavascriptコードと、いくつかのライブラリで構成。
  • 埋め込み : Webページやアプリに、あなた独自のビジュアルコードエディタを埋め込み。
  • オープンソース : MITライセンスで提供。

Download (Beta ver0.1)

source code on Github

デモ

編集ボタンをクリックすると、ビジュアルコードエディタを表示する。

実行ボタンをクリックすると、コードを実行する。

編集結果は、次のリストと実行内容に反映される。

  • 編集
  • 実行


		

		

		

			
やってみる

			
			
1.ダウンロード

			
次のファイルをダウンロードし、解凍する

			 Download
			
			
2.ファイルを開く

			
index.ja.htmlファイルを開くと、これと同じページが表示される。

			
ビジュアルコードエディタに表示される内容は、mameBlock_template.ja.htmlファイルを読み込んでいる。

		

		

		
	
			
導入する

			
あなた自身のHTMLファイルに次の内容(css、HTML、Javascript)を設定する。

			
			
1.CSS

			
スタイルシートを指定する

			
<link rel="stylesheet" type="text/css" href="css/mameblock.min.css" media="all" />

			
			
2.HTML

			
編集ダイアログを呼び出すボタンを設置

			
<a id="mameEdit" href="#">Edit</a>

			
			
ダイアログの枠組みを設置

			
<!-- edit dialog -->
<div id="modal-window">
	<a id="editor_close" href="#">[OK]</a>
	<div id="loading_area"></div>
</div>
<!-- express dialog -->
<div id="modal-express">
	<div class="modal-body">
		<label for="expModalText">Edit expression</label><br />
		<input id="expModalText" type="text" />
	</div>
	<a class="modal_close" href="#">[OK]</a>
</div>
			

			
			
3.Javascript

			
jqueryとライブラリを読み込む

			
<!-- Javascript -->
<script type="text/javascript" src="js/jquery-1.11.2.min.js"></script>
<script type="text/javascript" src="js/mameblock.min.js"></script>
			

			
			
制御スクリプトを読み込む

			
<script type="text/javascript">
$(function() {
    <!-- # Init -->
    var codeArea = $("#exampleArea");
    var codeText = MAME_BLOCK.init("#loading_area", "mameBlock_template.ja.html #mame_template");
    /* wait for DOM building */
    var wait0 = setTimeout(function() {
        setCode(MAME_BLOCK.getCodeBlock(), codeArea);
    }, 500);
    
    <!-- # Edit -->
    $("#modal-window").mameModal({
        closeOnEscape: false
    });

    $('#mameEdit').click(function(e){
        MAME_BLOCK.copyCodeBlock("#mame_template");
        $('#modal-window')
            .trigger('openModal')
            .css("top", $(window).scrollTop() + 10);
        e.preventDefault();
    });
    
    $('#editor_close').click(function(e){
        $('#modal-window').trigger('closeModal');
        MAME_BLOCK.reloadCodeBlock('#loading_area');
        setCode(MAME_BLOCK.getCodeBlock(), codeArea);
        e.preventDefault();
    });

});
</script>
			

			
4.その他

			
編集したコードのハイライト表示と実行には、好みのスクリプトを使用する。サンプルでは、highlight.jsと、Functionクラスを使っている。
